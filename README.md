# buddybot  
![doggo](imgs/doggo.jpg)

**Discord bot that answers like a nice dog!**

## .env  
Update the `.env` with your Discord bot token and the key from OpenAI  
- Token for the Discord bot: https://discord.com/developers/applications  
- OpenAI key: https://platform.openai.com/account/api-keys  

## hello bot  
To start the bot, run the following commands:

```bash
pip install -r requirements.txt
python discord_only.py
```

## buddy bot

Run the following commands:
```bash
pip install -r requirements.txt
python buddy_bot.py
```
